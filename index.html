The story

Sometimes you need to make a request to a premium third-party API, after a user input. Usually you fire the request on keyup. But each request costs ðŸ’µ ðŸ’µ ðŸ’µ and not every user is typing at the same speed.

Slow users cost you more money.
Because you are firing more and more requests while they are typing sloooooooooowly.

But what if...

We could learn how fast each user is typing and fire tailor made requests.

The solution

Howfast.js tracks every user keystroke and saves it localStorage. Then it calculates the last 100 tracked keystrokes and determines the average typing duration per user.
